Luas Segitiga

Rumus Luas segitiga :

L = 1/2 x a x t

L = Luas
a = Alas
t = Tinggi

Keliling Segitiga

Rumus Keliling segitiga :

K = A + B + C


Deskripsi:
K = Keliling
A = Sisi A
B = Sisi B
C = Sisi C
